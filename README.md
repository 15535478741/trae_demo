# trae_demo

资料真假性核验APP - AI驱动的资料来源识别与可信度评估平台

## 功能特性

- **文本检测**：输入文本进行真实性和准确性分析
- **链接验证**：粘贴网页链接检测来源可靠性
- **图片验证**：上传图片进行格式、元数据和编辑痕迹检测
- **文档检测**：支持PDF、Word、TXT、RTF等文档格式
- **AI内容检测**：识别AI生成内容，区分人类创作与AI辅助创作
- **可信度报告**：生成详细的分析报告和评分

## 技术栈

- React 18 + TypeScript
- Vite 6
- Tailwind CSS 3
- Zustand（状态管理）
- React Router DOM（路由）
- Lucide React（图标）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 部署到 GitHub Pages
npm run deploy
```
